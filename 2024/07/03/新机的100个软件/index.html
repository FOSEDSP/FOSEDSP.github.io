新电脑的100个软件

1.操作系统

用于下载Windows、Linux镜像

1.1 Windows镜像下载

  • 通过微软官网下载：Windows 11、Windows 10

  • MSDN, 我告诉你 - 做一个安静的工具站 (itellyou.cn)

  • 系统库－致力于原版windows生态服务 (xitongku.com)

  • 极简系统-最纯净的系统下载平台 (sysmini.com)

1.2 Linux镜像下载

  • 在Linux官网可以看到有24 个流行的 Linux 发行版下载链接
  • 国内开源镜像站：
    • 阿里巴巴开源镜像站
    • 清华大学开源镜像站
    • 网易开源镜像站

1.3 装机&虚拟机

  • U盘启动装机工具：**微PE工具箱、老毛桃、大白菜、Ventoy**

  • **Windows To Go**：便携版Windows

  • VMware Workstation: 功能强大的桌面虚拟机软件

  • VirtualBox: 免费开源的虚拟机软件

2.应用软件

2.1 浏览器

  • Google Chrome: 快速、稳定、安全，拥有丰富的扩展功能。

  • Mozilla Firefox: 开源且注重隐私保护，提供多样化的定制选项。

  • Microsoft Edge: 基于Chromium内核，集成Microsoft服务并优化性能。

  • 360极速浏览器: 基于Chromium内核，提供快速、安全的上网体验，并集成多种实用工具。

  • 星愿浏览器: 一款受到大学生喜爱的浏览器，具有漫画阅读模式等特色功能等，支持开放定制。

  • Vivaldi: 高度可定制，适合高级用户和开发者。

2.2 通信软件

  • **微信**：搭配微信多开、防撤回使用更佳
  • **QQ**：无需多言
  • **网易邮箱大师**：轻松管理多个邮箱

2.3 办公软件

  • Microsoft Office：下载、激活
  • **WPS Office**：教育考试专用版、还有各大高校定制版、政府定制版
  • **Canva可画**：制作PPT这个也不错

2.4 效率办公

  • 腾讯桌面整理：告别桌面杂乱无章，推荐下载独立版
  • PowerToys：微软开源工具，集成多种小工具

3.系统工具

3.1 解压软件

  • **7-Zip**：一款拥有极高压缩比的开源压缩软件，支持多种格式。
  • **WinRAR**：功能强大的压缩解压工具，支持多种格式，压缩率较高，操作简单。
  • **Bandizip**：一款免费的压缩解压软件，支持多种格式，界面简洁，操作方便。
  • **PeaZip**：跨平台的压缩解压软件，支持多种格式，提供了加密、分卷等功能。
  • **360压缩**：国产的压缩解压软件，支持多种格式，界面简洁，操作方便。

3.2 系统安全

  • **360安全卫士**：虽然大众对其争议比较大，但其杀毒能力还是毋庸置疑的，极速版可以考虑
  • **火绒安全**：拦截弹窗很棒，适合新手
  • 卡巴斯基：适合有重度需求的

3.3 软件卸载

  • **UninstallTool**：是一个小巧、安全、快速、强大的软件卸载删除工具

  • **Geek Uninstaller **： Windows 平台的专业卸载工具，体积小、操作简单、可强力卸载软件并清理注册表残留

  • **HiBit Uninstaller **：功能多样、小巧无广告的 Windows 软件卸载和系统清理工具

  • **Wise Force Deleter**：电脑中的文件删不了，不妨试试这个软件！

  • **IObit Unlocker**：文件解锁程序，电脑中的文件删不了，也可以试试这个！

  • **微软官方Office强力卸载工具**，轻松解决软件卸载残留问题

  • **Creative Cloud Cleaner Tool**：Adobe软件专业卸载工具

  • **AutoRemove**：需付费干净卸载Autodesk系列软件

3.4 电脑清理

  • **Dism++**：免费、开源的Windows系统优化工具，推荐！！

  • **CCleaner**：不适合小白，专业版激活

  • Wise Registry Cleaner：免费的注册表清理工具

  • **Registry Workshop**：简单易用的注册表编辑器

  • **360清理pro**：适合小白，一键清理，[网盘下载链接](https://wwb.lanzoue.com/i4Ly12302uqj
    密码:1314)

  • **Lenovo Quick Fix**：对比360清理pro，需要登录才能使用，有点恶心

  • 自动删除PC端微信大文件，不会删除聊天记录

3.5 系统维护

  • **[驱动精灵绿色版](https://wwb.lanzoue.com/i5siU2305wad
    密码:d3bo)**：没有胡乱绑定，没有胡乱下载 **万能网卡版**：适用无网络情况

  • **360驱动大师**：解决驱动安装更新问题

  • **DirectX修复工具**：C++运行库修复工具下载（通常报错msvcp100.dll、msvcp110.dll、msvcp120.dll、msvcp140.dll、msvcp140_1.dll、vcomp140.dll、vcamp140.dll修复一下就好）

  • **[微软常用运行库合集](https://wwb.lanzoue.com/iiQnD2307lmb
    密码:e4iy)**：微软常用运行库一键安装，可以解决缺少 dll 文件等问题

  • **傲梅分区助手**：无损分区、分区扩容、系统迁移

  • **CrystalDiskInfo**：HDD/SSD实用软件，检测硬盘信息等

  • **Windows Update Blocker**：一键禁用或启用Windows系统上的自动更新

  • **Defender Control v2.1**：一键开启或关闭Windows Defender

  • **Win11右键菜单样式切换**：轻松切换Windows 10 / 11右键单击上下文菜单

  • **TranslucentTB**：可以更改Windows任务栏透明度

  • **隔离沙盒**：安装可疑程序

4.个人信息管理（PIM）

4.1 任务管理

  • Microsoft To Do：微软推出，界面简洁，免费
  • 滴答清单 ：解锁高级版TickTick
  • eDiary ：非常轻便的日记本
  • 桌面日历：桌面日历形式的代办事项管理软件
  • any.do：跨平台同步

4.2 笔记和文档管理

  • 在线文档：石墨、腾讯文档、金山文档

  • all in one：Notion、飞书、**语雀**

  • [**Obsidian（黑曜石）**](Download - Obsidian)：结构化方式管理、本地存储，值得推荐

  • Typora：非常棒的一款 Markdown 编辑器和阅读器（89元）1.7.6版

  • **Zettlr**： 免费开源软件、适合写报纸文章、学期论文、论文或整本书

4.3 思维导图

  • Xmind：模版很丰富
  • **百度脑图**：在线工具，无需下载安装，方便快捷，基本功能齐全。
  • **FreeMind**：开源免费，适合个人使用。
  • 幕布：大纲笔记，一键生成思维导图 导出文档可使用幕布批量导出助手

4.4 流程图

  • Microsoft Visio：提供了丰富的图形和模板、与其他微软 Office 软件无缝集成，正版价格较高
  • **draw.io **：完全免费，Visio平替
  • 在线作图：Excalidraw、tldraw、Handraw | 中文友好的手写效果白板工具、亿图图示

截图工具

  • Snipaste：截图、贴图、标注；免费、免安装
  • PixPin ：需要安装，在Snipaste功能基础上，还支持长截图，截动图，文本识别。
  • QQ截图：需要登录，也可选择QQ截图独立版
  • ScreenToGif ：GIF录制
